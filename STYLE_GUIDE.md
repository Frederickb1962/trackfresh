# TrackFresh Style Guide v1

Living document — the landing page sets the pace for all other pages.

## Colors (LOCKED IN)
- Background (page): #064e3b (deep forest green — Tailwind emerald-900)
- Primary text: White (#FFFFFF)
- Section headings: Gold/amber — uppercase with letter-spacing
- Accent (CTA): Orange (Launch button style)
- Highlight text: Light/bright green (WASTE LESS. EAT BETTER. eyebrow)
- Feature callout: Gold (Tap to see what TrackFresh will do.)

## Typography
To be defined based on actual landing page values.

## Layout Patterns
- Mobile-first, single column
- Generous vertical spacing between sections
- Content padded 1rem on the sides
- Bullets: emoji prefix then short statement

## Voice & Copy Rules
- Short. Snappy. Confident.
- Use em-dashes for rhythm
- Active voice, verb-forward
- Avoid filler like "you can", "will be able to", "that is"
